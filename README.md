# Abaixo da FIPE — site

Páginas públicas do produto **Abaixo da FIPE** (landing + política de privacidade),
hospedadas via GitHub Pages.

- `index.html` — landing page (CTAs → checkout Kiwify)
- `politica-privacidade.html` — política de privacidade (exigida pelas lojas de extensão)

O código da extensão fica em repositório **privado** separado. Aqui só entra
conteúdo público. Ao editar a landing/política no repo da extensão
(`abaixo-da-fipe/marketing/`), copiar os arquivos atualizados pra cá e dar push.
